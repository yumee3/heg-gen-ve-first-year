### 2. GarageDoorSimulation

A Packet Tracer project demonstrating a simple home-automation system:

* Garage door opening and closing mechanism
* Sensors and actuators
* Manual and automatic trigger modes
* Basic IoT logic implemented within Packet Tracer

---

### 3. CiscoNetwork_All-in-One

A complete enterprise-style network configuration including:

* VLAN creation and segmentation
* Router-on-a-stick (inter-VLAN routing)
* DHCP server and DHCP relay configuration
* NAT, PAT, and static/dynamic translation rules
* ACL filtering (standard and extended)
* RIP routing configuration
* Wi-Fi SSID and WPA2 setup
* Clean IP addressing scheme and organized topology

## Purpose of This Repository

This repository serves as:

* A structured archive of networking laboratory work
* A reference toolkit for learning and revising Cisco IOS
* A practical resource for Packet Tracer-based exams

All `.pkt` files are intended for Cisco Packet Tracer (version 8.0 or later).
